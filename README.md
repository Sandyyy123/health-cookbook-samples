# Health & Nutrition Cookbook Samples

Structured cookbook writing samples for health-conscious adults. Each project follows a strict template system — consistent recipe format, nutrition notes, meal plans, food swap guides, and grocery organization.

## Books in This Repo

| Book | Focus | Target Reader |
|------|-------|---------------|
| [Liver-Friendly Kitchen](./liver-friendly-cookbook/) | Liver health, anti-inflammatory, low-sodium | Adults 45+ managing liver concerns |

## Template System

Every recipe entry follows the same structure:

```
1. Recipe Header (name, prep time, cook time, servings, difficulty)
2. Why It Works (1-2 sentences: what makes it liver-friendly)
3. Ingredients (affordable, grocery-store accessible)
4. Instructions (numbered, beginner-friendly, clear)
5. Nutrition Note (key benefit explained simply, no jargon)
6. Swap Guide (easy substitutions for dietary needs)
```

Meal plans, grocery lists, and food swap charts follow equally strict repeatable formats.

## Writing Approach

- All nutrition guidance cross-referenced against established dietary sources
- Reading level: accessible to adults with no medical background
- Tone: warm, supportive, practical — never clinical or fear-based
- AI-assisted drafting with human editing, accuracy review, and consistency QC
- Delivered in Google Docs or Word, formatted to client spec
