# The Liver-Friendly Kitchen

**Practical Meals and Nutrition Guidance for Adults 45+ Managing Liver Health**

A structured cookbook combining easy recipes, meal plans, food swap guides, and supportive wellness content — written for everyday people, not medical professionals.

## Book Structure

- **Introduction: Understanding Liver-Friendly Eating** (8 pages)
- **What to Eat, Limit, and Avoid** — food reference chart (6 pages)
- **Recipe Chapters by Category** (80 pages)
  - Breakfasts (10 recipes)
  - Light Lunches (10 recipes)
  - Hearty Dinners (12 recipes)
  - Snacks & Sides (8 recipes)
  - Drinks & Tonics (5 recipes)
- **4-Week Meal Plan** (12 pages)
- **Grocery Lists by Week** (8 pages)
- **Smart Food Swap Guide** (6 pages)

## Each Recipe Contains

1. Recipe name + prep/cook time + servings + difficulty
2. "Why It Works" — 1-2 sentences on the liver-friendly benefit
3. Ingredients (affordable, grocery-store accessible)
4. Step-by-step instructions (beginner-friendly)
5. Nutrition Note (key benefit, no jargon)
6. Easy Swap (substitutions for common dietary needs)
